Why?

사진 편집 어플 안의 콜라주 기능을 쓰다가 한계에 부딪혀서 기획해보았습니다.

function 1

사진 수 무제한

사진 편집 시 넣을 수 있는 사진 수를 많이 넣을 수 있도록 설정하여 제약없이 하나의 콜라주를 만들 수 있도록 합니다.

콜라주 결과물의 크기 직접 정하기

function 3

콜라주 테마

단순한 네모 모양이 아닌 우리나라 지도가 그려진 콜라주 편집 틀을 이용할 수 있도록 합니다. 각 지역에서 찍은 사진을 쉽게 넣을 수 있는 장점이 있습니다.
